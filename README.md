## 利用方法
アカウント作成 or ログインをするとホーム画面にみんなの投稿が表示されます。
フォームにAmazonのURLを入力すると商品画像も取得できます。

## 機能一覧
#### 投稿に関する機能
- ユーザー登録・ログイン(JWT)
- 投稿機能
    - アマゾンURLから商品画像を取得
- いいね機能
- 自分の投稿の削除機能
- 何時間前、何ヶ月前に投稿したか表示(timeago.js)
- 商品画像をクリックするとAmazonの商品ページを表示

#### ホーム
- 新規の５ユーザーを表示
- 新規投稿順に投稿を表示

#### プロフィールページ
- 背景画像ピッカー(pixabayAPI)
- Emojiピッカー(emoji-mart)
- 自分の投稿を新規順に表示
- 投稿数を表示
- フォロワーを表示

#### お気に入りページ
- いいねした投稿を新規順に表示

#### フォローページ
- フォローしているユーザーとその投稿を新規順に表示
